# Gauge Grid 马甲包复核

## 1. 马甲包功能

- **产品定位（中文）**：本地 **方格量测垫**：在可调整 N×N 的 **方格** 上为每格填数值或备注，**保存为命名 Board** 以便复用，并提供 **mm/cm/in 长度换算**；数据仅存本机，不依赖业务后端。
- **核心实体（中文）**：**Board（已保存的格子快照）**、**活网格（当前编辑区）**、**单位与精度设置**（默认长度单位、小数位数、网格边长 4–12）。

**除设置外至少三个可见主区块**（与底栏一致，便于扫一眼即见）：

1. **Grid（底栏「Grid」）**：主编辑区，显示 **N×N 宫格**；**点格** 弹出输入；可 **Clear all cells**、**Save board** 命名保存。
2. **Boards（底栏「Boards」）**：已存 **Board 列表**；**点行** 确认后**加载到活网格**（会覆盖当前活网格中未通过「再保存」的格子）；**左滑/侧滑** 可删除已存 Board。
3. **Convert（底栏「Convert」）**：**长度换算**；「From / To」单位 + 数值输入；**默认「From」单位** 与 **小数位数** 在 **Settings** 中可配。

- **Settings（AppBar 齿轮，全屏）**：**格宽（4–12）**、**默认长度单位**（mm/cm/in）、**换算显示小数位**、**擦除本机全部数据**、About；**不计入**底栏三格。

- **与 boot/remote 的衔接（中文）**：启动为 `app_common` 的 `BootPage → BootCoordinator → RemoteConfigClient`；`remoteConfigEndpoint` 即下文 **`remote_url` 的完整地址**；返回 **JSON 数组，取首元素** 按随机键解析。现网 [MockAPI 1qkp5](https://69ee1d1e9163f839f892848d.mockapi.io/1qkp5) 常见为 **`[]`**，则应用留在**本地**主壳。本地状态存于应用文档目录 `gauge_grid_state.json`。

## 3. 文生图 App Icon 提示词

**本包选用生肖吉祥物：兔（Rabbit, 兔）**；3D 卡通 chibi 风格、主体满框、1:1 方图、无字无商标；可握卷尺呼应「测量」；横版出图请 **内接方裁、勿大 letterbox** 再 1024²。

**提示词 A（推荐）**  
> 手机应用图标，1:1 方形，3D 卡通 chibi 生肖**兔**为主角，大眼镜神、软毛，手里拿黄色软尺，**满框**、少焦外留白；高对比、无字、无商标。若出横图，居中取最大正方再缩 1024，勿上下大灰边。

**提示词 B（英 brief 对齐）**  
> iOS app icon, square, cute 3D chibi **zodiac rabbit**, holding a measuring tape, full-bleed, soft bokeh, no text, no logo, legible at 64px; center-crop to square if export is 16:9, then 1024.

> 实装主图经 `sips` 对宽屏做 **H×H 内接方裁** 后 `sips -z 1024 1024`；母图可另存为 `assets/app_icon_source_wide.png` 备查。
