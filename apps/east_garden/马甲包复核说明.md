## 1. 马甲包功能

**East Garden**（显示名）定位为偏「桌边参考 + 轻量演练」的离线工具：界面与文案**全英文**，核心数据为本地「手牌速写」实体（最多 14 张牌位串）与 Courtyard 偏好，使用 `shared_preferences` 持久化，不依赖业务后端。

启动后先进入 `Xvhrt0` 门闸（与 `app_common` Boot 链等价的远程首包驱动 + Rule A 离线/联网补拉逻辑在生成文件内）。首包 JSON 数组取**第一个对象**；字段名为随机前缀 `xvhrt*`（见第 2 节）。`xvhrtPlaf` 为 `"1"`/`"2"`/`"3"` 时分别进入应用内 Web 容器两类或外开浏览器；缺失或空 URL 则进入本地五栏壳。应用内**不出现** `remote_url`、`MockAPI`、endpoint 等用户可见文案。

**除「Courtyard（设置）」外至少五个可见主区块（底栏直达）：**

| 名称 | 进入方式 |
|------|----------|
| Winds | 底栏第 1 项 |
| Honors | 底栏第 2 项 |
| Sketch | 底栏第 3 项（调色板 + 速写条，可保存） |
| Library | 底栏第 4 项（已存速写列表） |
| Bloom | 底栏第 5 项（可调时长的安静环） |

**Courtyard**：顶栏右侧自然风图标进入；含产品向设置（台布色盘、荣誉字重、 Bloom 时长、默认座位与 ribbon 等）与资料操作（剪贴板清单、清空库）。

**详情/编辑**：从 Library 点击某条进入 `HandEditorPage`，可改标题与拍面组合并写回库。

- **图源**：仓库内 `icon/icon_1.png`，应用内备份为 `assets/app_icon_source_master.png`，展示用 `assets/app_icon.png`（经 `sips` 缩放到 1024×1024，无上下黑边 letterbox）。

**Promotional Text**

Honor guides, wind notes, and tactile sketches share one tranquil studio. Library saves rehearsals; Bloom gently paces breathing pauses.

**Description**

East Garden is an English-only rehearsal studio for anyone who loves disciplined tile play. Start on Winds to revisit how seats and round stories interact, then open Honors for a friendly field guide to wind and dragon markers. Sketch lets you stage up to fourteen tiles from characters, dots, bamboo, and honors, then save snapshots that land instantly inside Library for later tuning. Bloom offers a simple breathing ring whose length you set in Courtyard, making it easy to cool down between drills. Courtyard itself holds the creative controls: felt-wash colors, honor typography, chip spacing, seat defaults, exportable sketch lists, and a safe wipe when you want a blank archive. Everything persists on-device, translations stay out of your way, and nothing in the interface nags you about configuration servers—just your table imagination.
