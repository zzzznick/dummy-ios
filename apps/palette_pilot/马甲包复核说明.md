# Palette Pilot（马甲包复核说明）

## 1. 马甲包功能

**产品定位**：离线配色工具（Color Lab），用于创建配色方案、做对比度校验、混色、查看色板并导出文本，全部本地可用，不依赖后端。

**核心实体**：Palette（调色板），包含名称与若干颜色（ARGB int 列表），持久化存储在本地。

**主要页面与交互**：
- 新建/编辑调色板：支持改名、添加颜色、调色（HSL）、调整顺序、删除颜色、保存持久化
- 对比度校验：选前景/背景色预览并计算对比度，按 AA/AAA 目标给出 pass/fail
- 混色：两色插值混合，支持把混合结果追加到已有调色板或新建一张调色板
- 色板：网格展示所有颜色，点击复制（HEX/RGB/HSL）文本
- 导出：生成文本导出（可复制），以及导出预览卡片（受设置里的 padding 影响）

**Boot/Remote 衔接**：
- App 启动先进入 `BootPage`，通过 `app_common` 的 `BootCoordinator → RemoteConfigClient` 拉取 `remote_url`。
- 即使远端返回空或失败，App 仍可正常进入本地功能（离线优先）。

**除设置外 ≥5 个可见主区块（进入方式）**：
1. **Palettes**：底部导航第 1 个 Tab
2. **Contrast**：底部导航第 2 个 Tab
3. **Mixer**：底部导航第 3 个 Tab
4. **Swatches**：底部导航第 4 个 Tab
5. **Export**：底部导航第 5 个 Tab

（Settings 为独立页面，通过各页右上角按钮进入，不计入以上 5 个主区块）

## 3. 文生图 App Icon 提示词

本包 App Icon 吉祥物：**兔（Rabbit）**。

要求：**偏卡通 / 可爱 3D chibi**，**1:1 方图**，**主体满框（约 80%+ 画幅）**，少留白，**无字无商标**，小尺寸（~64px）仍清晰；可让兔子拿“调色盘/画笔”等道具，但**动物为主**。若出图为横图/宽屏，落地时必须**内接正方居中裁剪**到 1024×1024，**禁止**上下大 letterbox 条。

可复制提示词（中文）：
1) `可爱 3D chibi 卡通兔子，正面半身，拿着小调色盘，主体满框 1:1 方形构图，背景极简柔和渐变，颜色高对比，小尺寸清晰，无文字无商标，适合作为 App icon`
2) `软萌 Q 版兔子吉祥物（单一物种），与配色工具道具互动（调色盘/色卡），1:1 方图，主体占画面 80%+，少留白，边缘安全适配 iOS 圆角，无字无 logo，禁止上下 letterbox`
