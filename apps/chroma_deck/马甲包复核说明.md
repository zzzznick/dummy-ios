# 马甲包复核说明（Chroma Deck — `chroma_deck`）

## 1. 马甲包功能

- **产品定位（中文）**：「Chroma Deck」是一款面向界面试计的**离线双配色工作区**：用「前景色 / 背景色」成对保存，方便对比、复制和检查对比度。核心数据是用户自定义的**配色对**（名称 + 两个颜色值），不依赖登录与后端业务接口。
- **主要页面 / 主区块（除设置外至少三个顶层）**：
  - **Deck（底栏第一项）**：以网格（2 列或 3 列，可在设置中切换）展示已保存的配色对；每张卡左右分栏。开启「对比度显示」时显示 **WCAG 对比度** 数值。
  - **Playground（底栏第二项）**：不依赖先保存的**实时**双配色大预览、实时对比度；可一键将当前两色**另存**为新 pairing（进 Deck）。
  - **Reference（底栏第三项）**：纯文案的 **WCAG 对比度说明**（AA/AAA 等），作为第三个独立主界面。
  - **Editor（全屏子页）**：新建/编辑单条 pairing（命名、两色、复制、保存、删除），从 Deck 点卡片或 AppBar 的 Add 进入。
  - **Settings（全屏子页）**：剪贴板格式、网格列数、是否在卡片上显示对比度、About。通过 AppBar 齿轮进入，与上面三档底栏**并列**为独立全屏，不计入底栏三格。
- **启动页可见性说明**：`remote_url` 若很快返回空数组，原先会在极短时间内就 `pushReplacement` 到主页，**肉眼几乎看不到** Boot 动画。当前实现为：Boot 上增加**英文提示文案**，并在首帧后 **约 0.9 秒** 再开始拉配置并导航，保证能看到启动布局（仍满足 `jacket-app-full-build`：非「仅一行 Booting」的敷衍页）。
- **与 boot / remote 的衔接**：
  - 启动时经 `app_common` 的 `BootPage → BootCoordinator → RemoteConfigClient` 拉取 `remote_url` 返回的 **JSON 列表**，取**第一个对象**作为 remote-config；根据其中的 `url` / `platform` 等字段由默认策略决定进本地 App 或壳页等（详见 `app_common`）。
  - **本包本地功能（配色存储、UI）完全可不依赖该接口**；当远端返回空列表或没有可用 URL 时，会落在**本地主界面**（当前你提供的 `api2` 往往返回 `[]`，适合本地联调与上架前占位）。
- **持久化**：应用文档目录下 `chroma_deck_state.json` 存配色列表与轻量设置；全英文产品界面与 `README` 以英文写清产品说明。

## 2. `remote_url` / 端点定义

- **本包常量**：`lib/boot/remote_config_endpoint.dart` 中 `remoteConfigEndpoint`。
- **完整 endpoint 字符串（须与实现一致、便于单文件核对）**：

`https://69e1e92fb1cb62b9f31779a9.mockapi.io/api2`

- **字段 keyset 说明**：`lib/boot/remote_config_keys.dart` 中 `remoteConfigKeys` 由 `tools/generate_remote_config_keyset.dart` 生成**随机 JSON 键名**（本包为 `qhvxm*` 前缀），与下述映射、响应示例一一对应；`BootPage` 内用该 endpoint + `remoteConfigKeys` 创建 `RemoteConfigClient`，供 `BootCoordinator` 拉取。改环境时同步 `remote_config_endpoint.dart` / `remote_config_keys.dart` 与 MockAPI 字段。

以下两段与仓库内 **`README.md` 的 Remote config 节** 内容一致，便于**仅打开本文件**即配置 MockAPI / 核对随机字段；接口返回为 **JSON 数组，取第一个对象** 生效。

## 3. 文生图 App Icon 提示词

以下用于贴到文生图模型，生成 **App 主图标**（与 `assets/app_icon.png` 设计意图一致，便于与英文 brief 交叉复核）。

**提示词 A（主推荐）**  
> 手机应用图标，1:1 圆角方形容器内，中心为两块圆角矩形色块交叠，一块深靛蓝（#3F51B5），一块暖珊瑚红（#E57373），扁平适量风格，轻阴影，高对比、无文字、无商标、无水印，背景极浅冷灰，小尺寸可辨。

**提示词 B（更强调几何）**  
> App icon, flat vector, two rounded squares overlapping, deep indigo and warm coral, subtle drop shadow, centered composition, no text, no letters, high legibility at 64px, light neutral background.

> 与 **step 3 英文 icon brief** 的对应：English brief 描述「indigo + coral、flat vector、无文字、小尺寸可辨」；上列中文 A 与 B 为同一产品「双色叠卡」的表述变体，供中文复核与文生图工具使用。
