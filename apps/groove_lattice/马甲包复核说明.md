# Groove Lattice 马甲包复核说明

## 一、马甲包功能

**产品定位：** 本地离线的多重节奏编排与练习工具（非日记/打卡类），聚焦于「多套声部脉冲如何在同一节拍窗口里对齐」的表达与试听感知的可视化。

**核心实体：** Lattice（Groove）：名称、BPM、`steps`（多声部的脉冲密度向量），保存在本机 SharedPreferences JSON 列表中。

**与 boot / remote 的衔接：**  
首启由命名空间组件 `lib/_krmwt/_krmwt.dart` 拉取远端 JSON 数组并仅使用第一项；字段名为随机前缀 `krmwt*`（与 README 中 Mapping 一致）。根据 `krmwtPlaf` 决定进入内置 Web 容器 / 外开浏览器 / 留在本地壳；无网时按规则 A 直接进本地壳并在恢复网络后一次性补拉。业务上的节奏数据、设置项**不依赖**任何后端。

**除「设置」外至少五个可见主区块（含进入方式）：**

1. **Studio** — 底栏「Studio」：当前 Groove 下拉、Lattice 舞台动画、元信息卡。  
2. **Composer** — 底栏「Composer」：三种节奏配方芯片、按 Settings 默认生成、空白编辑器入口。  
3. **Library** — 底栏「Library」：已存 Groove 列表，点行进入 **Lattice editor**（详情/编辑页）。  
4. **Drill** — 底栏「Drill」：点击大垫收集间隔、显示滚动平均与离散度。  
5. **Lab** — 底栏「Lab」：双环比例显微与「再次对齐所需主拍数」说明。

**设置：** 顶栏右侧图标进入 **Groove settings**（非底栏项），含：视觉模式分段按钮、默认 BPM/声部数、发光开关、种子风味下拉、横向色条换肤、一键清空库。

## 三、文生图 App Icon 提示词

**选用吉祥物物种：** **牛 / 役牛（Ox）** — **硬规范**：吉祥物**仅允许**从 **虎 / 牛 / 兔 / 鼠 / 龙** 中择一；本包为 **牛**，与行军鼓、鼓槌节奏主题一致。

**本包画风轮盘预设：** **(1) Soft 3D / chibi resin** — **软糯 3D 搪胶感 / Q 版立体角色**：柔顺布光与体积感，偏影视级卡通着色（非写实畜牧照片）；行军乐队青色制服 + 金饰、**黑白花牛**面孔与小角、纯色青绿背景，**正方形 1024**，无字无商标。

**当前上线素材说明：** Launcher 所使用的 `app_icon.png` 已由 **产品方供图**（同内容备份见 `assets/app_icon_source_master.png`），以下为若需复原或迭代的参考中文提示：

1. 软 3D、Q 版**奶牛/役牛**鼓手吉祥物，大眼睛、粉鼻、开怀笑，头侧**小弯角**与高帽白羽；**青色**仪队服装、**黄底青纹小军鼓**与**金色鼓槌**；纯色**青绿色**背景；**1:1 满构图**胸像，主体约占画面大部分；无文字/logo/水印；小图标轮廓清晰。

2. 若母图为横屏：须 **最大内接正方居中裁切** 后统一为 **1024×1024**，**禁止**上下大 letterbox 细条作为主 icon。
